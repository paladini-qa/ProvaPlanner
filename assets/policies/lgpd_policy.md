# Política de Processamento de Dados - LGPD

**Última atualização:** 26 de novembro de 2025  
**Versão:** 1.0.0

## 1. Conformidade com a LGPD

O ProvaPlanner está comprometido em cumprir a **Lei Geral de Proteção de Dados (Lei nº 13.709/2018)** e garantir que seus dados pessoais sejam tratados de forma transparente, segura e em conformidade com a legislação brasileira.

## 2. Base Legal para o Processamento

Processamos seus dados pessoais com base nas seguintes bases legais:

### 2.1 Consentimento

- Você fornece consentimento explícito ao aceitar esta política
- Você pode revogar seu consentimento a qualquer momento
- O consentimento é específico, informado e inequívoco

### 2.2 Execução de Contrato

- Processamos dados necessários para fornecer os serviços do aplicativo
- Dados necessários para cumprir obrigações contratuais

### 2.3 Legítimo Interesse

- Melhorar nossos serviços e experiência do usuário
- Garantir segurança e prevenir fraudes
- Análise estatística e desenvolvimento de funcionalidades

## 3. Finalidade do Processamento

Processamos seus dados pessoais para as seguintes finalidades:

### 3.1 Fornecimento do Serviço

- Gerenciar suas provas, revisões e metas de estudo
- Sincronizar dados entre dispositivos (quando aplicável)
- Personalizar sua experiência no aplicativo

### 3.2 Melhoria do Serviço

- Analisar padrões de uso para melhorar funcionalidades
- Desenvolver novas funcionalidades
- Corrigir bugs e problemas técnicos

### 3.3 Comunicação

- Enviar notificações sobre provas e revisões (se habilitado)
- Informar sobre atualizações importantes do aplicativo

## 4. Dados Pessoais Processados

### 4.1 Dados Identificadores

- Nome
- Email (quando você cria uma conta)
- Foto de perfil (opcional)

### 4.2 Dados de Uso

- Provas cadastradas (nome, data, disciplina)
- Revisões e metas de estudo
- Preferências e configurações
- Dados de uso do aplicativo

### 4.3 Dados Técnicos

- Informações do dispositivo
- Sistema operacional
- Versão do aplicativo

## 5. Compartilhamento de Dados

### 5.1 Sem Compartilhamento com Terceiros

**Não compartilhamos seus dados pessoais com terceiros**, exceto:

- Quando você nos dá permissão explícita
- Para cumprir obrigações legais
- Para proteger nossos direitos legais

### 5.2 Prestadores de Serviços

Podemos usar prestadores de serviços terceirizados para:

- Hospedagem de dados (quando aplicável)
- Análise de dados (anônimos e agregados)
- Suporte técnico

Esses prestadores são contratualmente obrigados a proteger seus dados.

## 6. Retenção de Dados

### 6.1 Período de Retenção

- Mantemos seus dados enquanto você usar o aplicativo
- Após exclusão da conta, dados são removidos em até 30 dias
- Dados podem ser mantidos por períodos mais longos se exigido por lei

### 6.2 Exclusão de Dados

Você pode solicitar a exclusão dos seus dados a qualquer momento através das configurações do aplicativo.

## 7. Seus Direitos (LGPD)

Conforme a LGPD, você tem os seguintes direitos:

### 7.1 Direito de Acesso

- Solicitar informações sobre quais dados pessoais processamos
- Obter uma cópia dos seus dados pessoais

### 7.2 Direito de Correção

- Corrigir dados incompletos, inexatos ou desatualizados
- Atualizar suas informações através das configurações

### 7.3 Direito de Exclusão

- Solicitar a exclusão dos seus dados pessoais
- Excluir sua conta e todos os dados associados

### 7.4 Direito à Portabilidade

- Receber seus dados em formato estruturado e interoperável
- Transferir seus dados para outro serviço

### 7.5 Direito de Revogação

- Revogar seu consentimento a qualquer momento
- Isso não afeta a legalidade do processamento anterior

### 7.6 Direito de Oposição

- Opor-se ao processamento de dados em certas circunstâncias
- Solicitar limitação do processamento

### 7.7 Direito de Revisão

- Solicitar revisão de decisões automatizadas
- Solicitar explicações sobre lógica de processamento

## 8. Segurança dos Dados

### 8.1 Medidas Técnicas

- Criptografia de dados sensíveis
- Armazenamento seguro
- Controle de acesso
- Backup regular

### 8.2 Medidas Organizacionais

- Políticas de privacidade internas
- Treinamento de equipe
- Auditorias regulares
- Procedimentos de resposta a incidentes

## 9. Transferência Internacional

Seus dados são processados principalmente no Brasil. Se houver transferência internacional, garantimos proteção adequada conforme a LGPD.

## 10. Dados de Menores

- Nosso aplicativo não é destinado a menores de 13 anos
- Não coletamos intencionalmente dados de menores
- Se você é menor de 18 anos, precisa de autorização dos pais/responsáveis

## 11. Encarregado de Proteção de Dados (DPO)

Para exercer seus direitos ou fazer perguntas sobre o processamento de dados, entre em contato através das configurações do aplicativo.

## 12. Alterações nesta Política

Podemos atualizar esta política periodicamente. Mudanças significativas serão notificadas e pode ser necessário novo consentimento.

## 13. Versão do Consentimento

Esta política está na **versão 1.0.0**. Quando houver atualizações, você será notificado e poderá revisar e aceitar a nova versão.

---

**Ao aceitar esta política, você consente com o processamento dos seus dados pessoais conforme descrito acima, em conformidade com a LGPD.**
